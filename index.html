TIC TAC TOE

Tic-Tac-Toe is a classic two-player game played on a 3x3 grid. Each player takes turns marking a square on the grid, one using "X" and the other using "O". The objective is to be the first to align three of their marks either horizontally, vertically, or diagonally. The game can end in a win for one player or in a draw if all squares are filled without a winner. Simple yet strategic, Tic-Tac-Toe is often used as a beginner's game or a quick challenge between friends.

CALCULATOR

A calculator app is a software application designed to perform arithmetic operations such as addition, subtraction, multiplication, and division. It typically features a user-friendly interface with a numeric keypad and buttons for basic mathematical functions. Some advanced versions may include additional features like scientific calculations, memory functions, and the ability to handle complex equations. Calculator apps are commonly used on smartphones, tablets, and computers to quickly solve mathematical problems in various settings, from everyday tasks to professional and educational purposes.
